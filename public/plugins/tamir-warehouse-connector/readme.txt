اتصال فروشگاه ووکامرس به پنل مدیریت انبار تامیر

== Description ==

این پلاگین امکان اتصال فروشگاه ووکامرس شما به پنل مدیریت انبار تامیر را فراهم می‌کند.

**ویژگی‌ها:**

* همگام‌سازی خودکار سفارشات جدید
* ارسال تغییرات وضعیت سفارش به پنل
* همگام‌سازی دستی سفارشات
* تست اتصال به پنل
* مشاهده وضعیت همگام‌سازی در صفحه سفارش

== Installation ==

1. فایل پلاگین را از پنل مدیریت انبار دانلود کنید
2. در پیشخوان وردپرس به "افزونه‌ها > افزودن" بروید
3. روی "بارگذاری افزونه" کلیک کنید
4. فایل zip پلاگین را انتخاب و نصب کنید
5. پلاگین را فعال کنید
6. به "ووکامرس > اتصال پنل انبار" بروید
7. آدرس پنل و کلید Webhook را وارد کنید
8. تنظیمات را ذخیره کنید

== Frequently Asked Questions ==

= آیا نیاز به تنظیمات خاصی در ووکامرس است؟ =

خیر، فقط کافیست آدرس پنل و کلید Webhook را در تنظیمات پلاگین وارد کنید.

= چگونه کلید Webhook را دریافت کنم؟ =

کلید Webhook در بخش تنظیمات پنل مدیریت انبار قابل مشاهده است.

= آیا سفارشات قبلی هم همگام‌سازی می‌شوند؟ =

بله، با استفاده از دکمه "همگام‌سازی دستی" می‌توانید سفارشات 7 روز اخیر را همگام‌سازی کنید.
